# 2026-03-20 - 量化交易工具调试

## 项目：量化交易工具 (quant-tool)

**部署地址：**
- 本地：http://localhost:3000
- Railway：https://quant-tool-production.up.railway.app

**GitHub：** https://github.com/moonwalk92/quant-tool

## 问题排查记录

### 问题 1：Railway 部署后无法访问
**原因：** 服务只绑定到 localhost，无法接受外部请求
**解决：** 修改 `src/server.js`，将 `server.listen(PORT)` 改为 `server.listen(PORT, '0.0.0.0')`

### 问题 2：服务重启后交易引擎状态异常
**原因：** 旧进程未被完全杀死，模拟交易继续运行并触发回撤限制
**解决：** 在服务启动时自动调用 `engine.reset()` 重置状态

### 问题 3：前端收不到实时价格数据（正在排查）
**现象：** WebSocket 连接后，前端不显示实时价格和权益
**已修复：**
- 改进 WebSocket 连接配置（增加重连机制）
- 添加详细调试日志
- 修复了 `updateUI` 函数重复定义的问题

**待验证：** 需要用户强制刷新浏览器（Cmd+Shift+R）查看 Console 日志

## 技术细节

**技术栈：**
- 后端：Node.js + Express + Socket.IO
- 前端：原生 HTML/JS/CSS
- 交易引擎：模拟 MT4 交易（未连接真实 MT4）

**风控参数：**
- 初始资金：$10,000
- 单笔风险：1%
- 最大回撤：20%
- 交易品种：XAUUSD（黄金）

**交易策略：**
- 双向多开策略
- 顺势挂单
- 自动止盈止损（20 点）
